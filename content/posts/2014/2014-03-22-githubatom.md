Github社製エディタAtomの使用感、少し使ってみて馴染んできたのでメモしたいと想います。

自身のエディタ変遷
---------------

自身のエディタ経緯

サクラ -> 秀丸 -> Eclipse -> coda -> sublime -> vim ときどき Xcode

と、なんというか進化しているのか退化しているのか微妙な流れ出は有りますが、折角だしAtom使ってみようと思います。


sublimeよりかゆいところに手が届く感じだと嬉しいなーと期待しつつ


ぱっと見
-------

sublime。

command paletteとかも

	cmd-shift-P
	
で似た感じ。

ちょっと嫌だなと思ったポイント
